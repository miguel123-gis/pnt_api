API for Philippine Native Trees

Still a work in progress!

http://pinoytrees.com/ntapi/


Sample endpoints:


name=Smooth%20Narra

species=Dipterocarpus validus

family=Lamiaceae


classification=endemic

status=vulnerable

Based on DENR Administrative Order 2017-11 which contains the "National List of Threatened Philippine Plants and their categories"

https://www.philippineplants.org/Resources/dao-2017-11.pdf


priority=yes

If the species is included in BINHI's priority species database

https://binhi.ph/trees/
